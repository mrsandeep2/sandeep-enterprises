# Sandeep Enterprises

Premium quality rice, atta, and cattle feed e-commerce platform.

## About

Sandeep Enterprises is a wholesale supplier of premium quality rice, wheat atta, kapila cattle feed, and chokar in India.

## Products

- **Rice**: Basmati Rice, Sona Masoori, Parmal Rice, Biryani Rice, Katarani Rice
- **Flour**: Wheat Atta, Chokar
- **Animal Feed**: Kapila Cattle Feed
